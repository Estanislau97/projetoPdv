# API PDV(PONTO DE VENDA) WEB
Está API PDV está em fase inicial e oferece funcionalidades básicas para cadastrar usuário, gerenciar transações de venda, e o estoque de produtos. Com o objetivo de explorar e aprimorar os meus conhecimentos em desenvolvimentos de software. 

## Ferramentas utilizadas até aqui
Linguagem de programação Java
Framework Spring Boot, JPA(Java Persistence API), Hibernate
Banco de Dados SQL com SGBD MySQL
Ferramenta para automação de compilação Maven, e o Postman para  realizar e testar requisições HTTP para a API, facilitando o desenvolvimento e a depuração dos endpoints.

## Funcionalidades Principais
Cadastro de Produtos: Permite adicionar, atualizar e remover produtos do estoque.
Gestão de Vendas: Registra novas vendas, permitindo a inclusão de itens, a atualização de quantidades e a remoção de itens.
Consulta de Estoque: Fornece informações sobre a quantidade disponível de cada produto no inventário.
Relatórios de Vendas: Gera relatórios básicos sobre as transações realizadas, incluindo data, valor total e itens vendidos.
